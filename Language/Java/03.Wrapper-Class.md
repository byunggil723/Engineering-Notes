# Wrapper Class 주요 메서드

### 1. 공통 메서드

#### 1\) `valueOf(...)`

문자열이나 primitive 값을 wrapper 객체로 만든다.

#### 숫자형
```java
Byte b1 = Byte.valueOf((byte) 10);
Byte b2 = Byte.valueOf("10");

Short s1 = Short.valueOf((short) 20);
Short s2 = Short.valueOf("20");

Integer i1 = Integer.valueOf(30);
Integer i2 = Integer.valueOf("30");

Long l1 = Long.valueOf(40L);
Long l2 = Long.valueOf("40");

Float f1 = Float.valueOf(3.14f);
Float f2 = Float.valueOf("3.14");

Double d1 = Double.valueOf(6.28);
Double d2 = Double.valueOf("6.28");
```

#### 문자형

```java
Character c1 = Character.valueOf('A');
```

#### Boolean

```java
Boolean bo1 = Boolean.valueOf(true);
Boolean bo2 = Boolean.valueOf("true");
```

#### 2\) `parseXXX(...)`

문자열을 primitive 값으로 바꾼다.

```java
byte b = Byte.parseByte("10");
short s = Short.parseShort("20");
int i = Integer.parseInt("30");
long l = Long.parseLong("40");
float f = Float.parseFloat("3.14");
double d = Double.parseDouble("6.28");
boolean bo = Boolean.parseBoolean("true");
// Character.parseChar(...)  // 없음
```

#### 3\) `toString(...)`

```java
Byte b = 10;
Short s = 20;
Integer i = 30;
Long l = 40L;
Float f = 3.14f;
Double d = 6.28;
Character c = 'A';
Boolean bo = true;

// 인스턴스 메서드
String sb = b.toString();
String ss = s.toString();
String si = i.toString();
String sl = l.toString();
String sf = f.toString();
String sd = d.toString();
String sc = c.toString();
String sbo = bo.toString();

// static 메서드
String sb2 = Byte.toString((byte) 10);
String ss2 = Short.toString((short) 20);
String si2 = Integer.toString(30);
String sl2 = Long.toString(40L);
String sf2 = Float.toString(3.14f);
String sd2 = Double.toString(6.28);
String sc2 = Character.toString('A');
String sbo2 = Boolean.toString(true);
```

#### 4\) `equals(...)`

같은 wrapper 객체 간 등호 비교

```java
Byte b1 = 10, b2 = 10;
Short s1 = 20, s2 = 20;
Integer i1 = 30, i2 = 30;
Long l1 = 40L, l2 = 40L;
Float f1 = 3.14f, f2 = 3.14f;
Double d1 = 6.28, d2 = 6.28;
Character c1 = 'A', c2 = 'A';
Boolean bo1 = true, bo2 = true;

b1.equals(b2);
s1.equals(s2);
i1.equals(i2);
l1.equals(l2);
f1.equals(f2);
d1.equals(d2);
c1.equals(c2);
bo1.equals(bo2);
```

#### 5\) `compareTo(...)`

같은 wrapper 객체 간 대소 비교

```java
Byte b1 = 10, b2 = 20;
Short s1 = 10, s2 = 20;
Integer i1 = 10, i2 = 20;
Long l1 = 10L, l2 = 20L;
Float f1 = 1.5f, f2 = 2.5f;
Double d1 = 1.5, d2 = 2.5;
Character c1 = 'A', c2 = 'B';
Boolean bo1 = false, bo2 = true;

b1.compareTo(b2);
s1.compareTo(s2);
i1.compareTo(i2);
l1.compareTo(l2);
f1.compareTo(f2);
d1.compareTo(d2);
c1.compareTo(c2);
bo1.compareTo(bo2);
```

#### 6\) `compare(...)`

같은 wrapper 객체 간 대소 비교 (static 메서드)

```java
Byte.compare((byte) 10, (byte) 20);
Short.compare((short) 10, (short) 20);
Integer.compare(10, 20);
Long.compare(10L, 20L);
Float.compare(1.5f, 2.5f);
Double.compare(1.5, 2.5);
Character.compare('A', 'B');
Boolean.compare(false, true);
```

#### 7\) `xxxValue()`

숫자형 wrapper 객체 안의 값을 primitive로 꺼냄

```java
Byte b = 10;
Short s = 20;
Integer i = 30;
Long l = 40L;
Float f = 3.14f;
Double d = 6.28;

b.byteValue();    b.shortValue();    b.intValue();    b.longValue();    b.floatValue();    b.doubleValue();
s.byteValue();    s.shortValue();    s.intValue();    s.longValue();    s.floatValue();    s.doubleValue();
i.byteValue();    i.shortValue();    i.intValue();    i.longValue();    i.floatValue();    i.doubleValue();
l.byteValue();    l.shortValue();    l.intValue();    l.longValue();    l.floatValue();    l.doubleValue();
f.byteValue();    f.shortValue();    f.intValue();    f.longValue();    f.floatValue();    f.doubleValue();
d.byteValue();    d.shortValue();    d.intValue();    d.longValue();    d.floatValue();    d.doubleValue();

Character c = 'A';
c.charValue();

// booleanValue() 없음
```

### 2. 진법 관련 메서드

주로 Integer, Long에 존재

#### 1\) `toBinaryString(...)`

10진수를 2진수 문자열로 변환

```java
Integer.toBinaryString(10); // "1010"
Long.toBinaryString(10L);   // "1010"
```

#### 2\) `toOctalString(...)`

10진수를 8진수 문자열로 변환

```java
Integer.toOctalString(10); // "12"
Long.toOctalString(10L);   // "12"
```

#### 3\) `toHexString(...)`

10진수를 16진수 문자열로 변환

```java
Integer.toHexString(10); // "a"
Long.toHexString(10L);   // "a"
```

#### 4\) `parseInt(..., radix)` / `parseLong(..., radix)`

특정 진법의 문자열을 10진수로 변환

```java
Integer.parseInt("1010", 2); // 10
Integer.parseInt("12", 8);   // 10
Integer.parseInt("a", 16);   // 10

Long.parseLong("1010", 2);   // 10
Long.parseLong("12", 8);     // 10
Long.parseLong("a", 16);     // 10
```

### 3. Max / Min / Sum

```java
Integer.max(10, 20);
Integer.min(10, 20);
Integer.sum(10, 20);

Long.max(10L, 20L);
Long.min(10L, 20L);
Long.sum(10L, 20L);

Float.max(1.5f, 2.5f);
Float.min(1.5f, 2.5f);
Float.sum(1.5f, 2.5f);

Double.max(1.5, 2.5);
Double.min(1.5, 2.5);
Double.sum(1.5, 2.5);
```

### 4. Character 관련 메서드

#### 1\) `isDigit(...)`

문자 `c`가 숫자 문자인지 검사

```java
Character.isDigit('3');   // true
Character.isDigit('A');   // false
Character.isDigit('가');  // false
```

#### 2\) `isLetter(...)`

문자 `c`가 문자(글자) 인지 검사

```java
Character.isLetter('A');   // true
Character.isLetter('z');   // true
Character.isLetter('가');  // true
Character.isLetter('3');   // false
Character.isLetter('%');   // false
```

#### 3\) `isLetterOrDigit(...)`

문자 `c`가 문자 또는 숫자인지 검사

```java
Character.isLetterOrDigit('A');  // true
Character.isLetterOrDigit('3');  // true
Character.isLetterOrDigit('가'); // true
Character.isLetterOrDigit('%');  // false
Character.isLetterOrDigit(' ');  // false
```

#### 4\) `isUpperCase(...)`

문자 `c`가 대문자인지 검사

```java
Character.isUpperCase('A'); // true
Character.isUpperCase('a'); // false
```

#### 5\) `isLowerCase(...)`

문자 `c`가 소문자인지 검사

```java
Character.isLowerCase('a'); // true
Character.isLowerCase('A'); // false
```

#### 6\) `isWhitespace(...)`

문자 `c`가 공백 문자인지 검사

```java
Character.isWhitespace(' ');   // true
Character.isWhitespace('\n');  // true
Character.isWhitespace('\t');  // true
Character.isWhitespace('A');   // false
```

#### 7\) `toUpperCase(...)`

문자 `c`를 대문자로 바꿔서 반환

```java
Character.toUpperCase('b'); // 'B'
Character.toUpperCase('A'); // 'A'

// 원래 문자를 직접 바꾸는 게 아니라, 변환된 결과를 반환
```

#### 8\) `toLowerCase(...)`

문자 `c`를 소문자로 바꿔서 반환

```java
Character.toLowerCase('Q'); // 'q'
Character.toLowerCase('m'); // 'm'
```
