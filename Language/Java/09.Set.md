# Set

### Set(HashSet)

```Java
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Set<Integer> set = new HashSet<>();
    // 중복 허용 X, 순서 보장 X
    // HashSet은 내부적으로 해시 기반으로 동작하며, 평균 O(1) 시간복잡도를 가진다.

    set.add(1); // 삽입
    set.add(2);
    set.add(2); // 중복 삽입 → 무시됨

    set.contains(1); // 값 존재 여부 확인 → true

    set.remove(1); // 삭제
    set.size(); // 크기
    set.isEmpty(); // 비어있는지 확인

    // 순회 (순서 보장 X)
    // 삽입 순서를 유지하고 싶은 경우, Set<Integer> set = new LinkedHashSet<>(); 로 선언
    for (int value : set) {
      System.out.println(value);
    }
  }
}
```

### TreeSet (정렬 Set)

#### 1\) 오름차순(기본)

```Java
import java.util.*;

Set<Integer> set = new TreeSet<>();
// 중복 허용 X, 자동 정렬
// TreeSet은 내부적으로 Red-Black Tree를 사용하여 자동 정렬되는 Map 구현체로, 
// O(log N) 시간복잡도를 가진다.

set.add(3);
set.add(1);
set.add(2);

set.contains(2);     // true

set.remove(3);       // 삭제
set.size();          
set.isEmpty();       

// 정렬된 상태로 반환
for (int value : set) {
  System.out.println(value); // 1 2
}
```

#### 2\) 내림차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Set<Integer> set = new TreeSet<>((a, b) -> {
      return Integer.compare(b, a);
    });

    set.add(3);
    set.add(1);
    set.add(2);

    set.contains(2); // true

    set.remove(3); // 삭제
    set.size();
    set.isEmpty();

    // 정렬된 상태로 반환
    for (int value : set) {
      System.out.println(value); // 2 1
    }
  }
}
```

### 3\) 복합 정렬

```java
import java.util.*;

class Pair {
  int x, y;

  Pair(int x, int y) {
    this.x = x;
    this.y = y;
  }
}

public class Main {
  public static void main(String[] args) throws Exception {
    TreeSet<Pair> set = new TreeSet<>((a, b) -> {
      if (a.y != b.y)
        return Integer.compare(a.y, b.y);
      return Integer.compare(b.x, a.x);
    });

    // 삽입
    set.add(new Pair(2, 3));
    set.add(new Pair(1, 5));
    set.add(new Pair(1, 2));
    set.add(new Pair(3, 1));
    set.add(new Pair(3, 7));
    set.add(new Pair(4, 1));
    set.add(new Pair(5, 6));
    set.add(new Pair(3, 2));

    // 존재 확인
    System.out.println(set.contains(new Pair(1, 2))); // true
    System.out.println(set.contains(new Pair(1, 3))); // false

    // 삭제
    System.out.println(set.remove(new Pair(1, 5))); // true
    System.out.println(set.remove(new Pair(1, 5))); // false

    // 출력
    for (Pair p : set) {
      System.out.println(p.x + " " + p.y);
    }

    /*
     * 4 1
     * 3 1
     * 3 2
     * 1 2
     * 2 3
     * 5 6
     * 3 7
     */
  }
}
```
