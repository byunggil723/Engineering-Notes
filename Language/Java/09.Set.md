# Set

### Set(HashSet)

```Java
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Set<Integer> s = new HashSet<>();
    // 중복 허용 X, 순서 보장 X
    // HashSet은 내부적으로 해시 기반으로 동작하며, 평균 O(1) 시간복잡도를 가진다.

    s.add(1); // 삽입
    s.add(2);
    s.add(2); // 중복 삽입 → 무시됨

    s.contains(1); // 값 존재 여부 확인 → true

    s.remove(1); // 삭제
    s.size(); // 크기
    s.isEmpty(); // 비어있는지 확인

    // 순회 (순서 보장 X)
    // 삽입 순서를 유지하고 싶은 경우, Set<Integer> set = new LinkedHashSet<>(); 로 선언
    for (int value : s) {
      System.out.println(value);
    }
  }
}
```

### TreeSet (정렬 Set)

#### 1\) 오름차순(기본)

```Java
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Set<Integer> s = new TreeSet<>();
    // 중복 허용 X, 자동 정렬
    // TreeSet은 내부적으로 Red-Black Tree를 사용하여 자동 정렬되는 Map 구현체로,
    // O(log N) 시간복잡도를 가진다.

    s.add(3);
    s.add(1);
    s.add(2);

    s.contains(2); // true

    s.remove(3); // 삭제
    s.size();
    s.isEmpty();

    // Set의 수정 메서드는 존재하지 않는다.
    // 특정 값을 지우고, 새로운 값을 추가하는 방식으로 처리한다.

    // 정렬된 상태로 반환
    for (int value : s) {
      System.out.println(value); // 1 2
    }
  }
}
```

#### 2\) 내림차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Set<Integer> s = new TreeSet<>((a, b) -> {
      return Integer.compare(b, a);
    });

    s.add(3);
    s.add(1);
    s.add(2);

    s.contains(2); // true

    s.remove(3); // 삭제
    s.size();
    s.isEmpty();

    // 정렬된 상태로 반환
    for (int value : s) {
      System.out.println(value); // 2 1
    }
  }
}
```

### 3\) 복합 정렬

```java
import java.util.*;

class Pair {
  int x, y;

  Pair(int x, int y) {
    this.x = x;
    this.y = y;
  }
}

public class Main {
  public static void main(String[] args) throws Exception {
    TreeSet<Pair> s = new TreeSet<>((a, b) -> {
      if (a.y != b.y)
        return Integer.compare(a.y, b.y);
      return Integer.compare(b.x, a.x);
    });

    // 삽입
    s.add(new Pair(2, 3));
    s.add(new Pair(1, 5));
    s.add(new Pair(1, 2));
    s.add(new Pair(3, 1));
    s.add(new Pair(3, 7));
    s.add(new Pair(4, 1));
    s.add(new Pair(5, 6));
    s.add(new Pair(3, 2));

    // 존재 확인
    System.out.println(s.contains(new Pair(1, 2))); // true
    System.out.println(s.contains(new Pair(1, 3))); // false

    // 삭제
    System.out.println(s.remove(new Pair(1, 5))); // true
    System.out.println(s.remove(new Pair(1, 5))); // false

    // 출력
    for (Pair p : s) {
      System.out.println(p.x + " " + p.y);
    }

    /*
     * 4 1
     * 3 1
     * 3 2
     * 1 2
     * 2 3
     * 5 6
     * 3 7
     */
  }
}
```
