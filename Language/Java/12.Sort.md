# 정렬

## 1. `Arrays.sort(...)`

유틸리티 클래스 `Arrays`에 속하는 메서드이다.
primitive 배열(`int[]`, `long[]`, ...)과 객체 배열(`Integer[]`, `String[]`, ...)을 정렬할 수 있으며,<br>
List 계열의 Collection(ArrayList<E>, LinkedList<E>, Vector<E>, Stack<E>)은 정렬 불가능하다.

### 사용 예시

#### 1\) primitive 배열 정렬

primitive 배열은 기본적으로 오름차순 정렬만 가능하다.<br>
이는 `Arrays` 내부에 `sort(int[])`는 구현되어 있지만, 객체 배열의 `sort(Object[], Comparator)` 형태와 같은 메서드는 제공되지 않기 때문이다.<br>
따라서 int[]를 내림차순으로 정렬하려면, 오름차순으로 정렬한 뒤 순서를 뒤집거나 직접 정렬 로직을 구현해야 한다.

#### 오름차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    int[] arr = { 5, 2, 8, 1, 3 };

    Arrays.sort(arr);

    System.out.println(Arrays.toString(arr)); // [1, 2, 3, 5, 8]
  }
}
```

#### 내림차순

```java
// 오름차순 정렬 후 뒤집기
import java.util.*;

public class Main {
    public static void main(String[] args) {
        int[] arr = {5, 2, 8, 1, 3};

        Arrays.sort(arr);

        for (int i = 0; i < arr.length / 2; i++) {
            int tmp = arr[i];
            arr[i] = arr[arr.length - 1 - i];
            arr[arr.length - 1 - i] = tmp;
        }

        System.out.println(Arrays.toString(arr)); // [8, 5, 3, 2, 1]
    }
}

// 내림차순 로직 구현
import java.util.*;

interface IntComparator {
  int compare(int a, int b);
}

public class Main {
  // 구현 예시용 O(N^2) 함수
  public static void sort(int[] arr, IntComparator comp) {
    for (int i = 0; i < arr.length - 1; i++) {
      for (int j = i + 1; j < arr.length; j++) {
        if (comp.compare(arr[i], arr[j]) > 0) {
          int tmp = arr[i];
          arr[i] = arr[j];
          arr[j] = tmp;
        }
      }
    }
  }

  public static void main(String[] args) {
    int[] arr1 = { 5, 1, 4, 2, 3 };
    int[] arr2 = { 5, 1, 4, 2, 3 };

    // 오름차순
    sort(arr1, (a, b) -> Integer.compare(a, b));
    /*
     * 람다가 특별 취급되는 것,
     * 컴파일러가 이걸 보고
     * (a, b) -> Integer.compare(a, b)
     * 를 그냥 함수처럼 넣는 게 아니라, IntComparator 자리에 들어가야 하니까
     * IntComparator 구현 객체 하나를 만들어 주는 것이다.
     */
    System.out.println(Arrays.toString(arr1)); // [1, 2, 3, 4, 5]

    // 내림차순
    sort(arr2, (a, b) -> Integer.compare(b, a));
    System.out.println(Arrays.toString(arr2)); // [5, 4, 3, 2, 1]
  }
}
```

#### 2\) 객체 배열 정렬

#### 오름차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    Integer[] arr = { 5, 2, 8, 1, 3 };

    Arrays.sort(arr);
    // 혹은, 직접 람다 함수 선언
    Arrays.sort(arr, (a, b) -> Integer.compare(a, b));

    // arr.sort(null); 불가, 객체 배열은 정렬용 인스턴스 메서드가 제공되지 않는다.

    System.out.println(Arrays.toString(arr)); // [1, 2, 3, 5, 8]
  }
}
```

#### 내림차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    Integer[] arr = { 5, 2, 8, 1, 3 };

    // 유틸리티 클래스 Collections 메서드 사용
    Arrays.sort(arr, Collections.reverseOrder());
    // 혹은, 직접 람다 함수 선언
    Arrays.sort(arr, (a, b) -> Integer.compare(b, a));

    System.out.println(Arrays.toString(arr)); // [8, 5, 3, 2, 1]
  }
}
```

## 2. `Collections.sort(...)`

유틸리티 클래스 `Collections`에 속하는 메서드이다.
List 계열의 Collection(ArrayList<E>, LinkedList<E>, Vector<E>, Stack<E>)을 정렬할 수 있으며,<br>
primitive 배열(`int[]`, `long[]`, ...)과 객체 배열(`Integer[]`, `String[]`, ...)은 정렬 불가능하다.

### 사용 예시

#### 오름차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    List<Integer> arr = new ArrayList<>(List.of(5, 2, 8, 1, 3));

    Collections.sort(arr);
    // 혹은, 직접 람다 함수 선언
    Collections.sort(arr, (a, b) -> Integer.compare(a, b));

    System.out.println(arr); // [1, 2, 3, 5, 8]
  }
}
```

#### 내림차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    List<Integer> arr = new ArrayList<>(List.of(5, 2, 8, 1, 3));

    // 유틸리티 클래스 Collections 메서드 사용
    Collections.sort(arr, Collections.reverseOrder());
    // 혹은, 직접 람다 함수 선언
    Collections.sort(arr, (a, b) -> Integer.compare(b, a));

    System.out.println(arr); // [8, 5, 3, 2, 1]
  }
}
```

## 3. `List.sort(...)`

List 인터페이스에 정의된 인스턴스 메서드이다.<br>
List 계열의 Collection(ArrayList<E>, LinkedList<E>, Vector<E>, Stack<E>)이 직접 메서드를 호출하여 정렬한다.<br>

### 사용 예시

#### 오름차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    List<Integer> arr = new ArrayList<>(List.of(5, 2, 8, 1, 3));

    arr.sort(null);
    // 혹은, 직접 람다 함수 선언
    arr.sort((a, b) -> Integer.compare(a, b));

    System.out.println(arr); // [1, 2, 3, 5, 8]
  }
}
```

#### 내림차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    List<Integer> arr = new ArrayList<>(List.of(5, 2, 8, 1, 3));

    arr.sort(Collections.reverseOrder());
    // 혹은, 직접 람다 함수 선언
    arr.sort((a, b) -> Integer.compare(b, a));

    System.out.println(arr); // [8, 5, 3, 2, 1]
  }
}
```

## 4. 복합 정렬

### 사용 예시

#### 1\) Pair 정렬

```java
import java.util.*;

class Pair {
  int x, y;

  Pair(int x, int y) {
    this.x = x;
    this.y = y;
  }
}

public class Main {
  public static void main(String[] args) throws Exception {
    // 객체 배열
    Pair[] arr = new Pair[] {
        new Pair(1, 2),
        new Pair(4, 3),
        new Pair(3, 6),
        new Pair(1, 10),
        new Pair(1, 5)
    };

    // 첫 번째 원소(우선 정렬): 오름차순, 두 번째 원소: 내림차순
    Arrays.sort(arr, (a, b) -> {
      if (a.x != b.x)
        return Integer.compare(a.x, b.x);
      else
        return Integer.compare(b.y, a.y);
    });

    /*
     * 1 10
     * 1 5
     * 1 2
     * 3 6
     * 4 3
     */

    // List 계열 Collection 생성
    List<Pair> list = new ArrayList<>();

    list.add(new Pair(1, 2));
    list.add(new Pair(2, 1));
    list.add(new Pair(4, 1));
    list.add(new Pair(3, 2));
    list.add(new Pair(5, 2));
    list.add(new Pair(1, 4));

    // 1) List 인터페이스 메서드 사용
    // 첫 번째 원소(우선 정렬): 오름차순, 두 번째 원소: 내림차순
    list.sort((a, b) -> {
      if (a.x != b.x)
        return Integer.compare(a.x, b.x);
      else
        return Integer.compare(b.y, a.y);
    });

    /*
     * 1 4
     * 1 2
     * 2 1
     * 3 2
     * 4 1
     * 5 2
     */

    // 2) Collections 유틸리티 클래스 메서드 사용
    // 두 번째 원소(우선 정렬): 오름차순, 첫 번째 원소: 내림차순
    Collections.sort(list, (a, b) -> {
      if (a.y != b.y)
        return Integer.compare(a.y, b.y);
      else
        return Integer.compare(b.x, a.x);
    });

    /*
     * 4 1
     * 2 1
     * 5 2
     * 3 2
     * 1 2
     * 1 4
     */
  }
}
```

#### 2\) Tuple 정렬

```java
import java.util.*;

class Tuple {
  int x, y, z;

  Tuple(int x, int y, int z) {
    this.x = x;
    this.y = y;
    this.z = z;
  }
}

public class Main {
  public static void main(String[] args) throws Exception {
    // 객체 배열
    Tuple[] arr = new Tuple[] {
        new Tuple(1, 2, 4),
        new Tuple(4, 3, 1),
        new Tuple(3, 6, 2),
        new Tuple(1, 10, 5),
        new Tuple(2, 5, 7),
        new Tuple(3, 6, 3),
        new Tuple(1, 10, 5),
        new Tuple(3, 5, 7),
        new Tuple(3, 6, 1),
        new Tuple(4, 10, 5)
    };

    // 첫 번째 원소(우선 정렬): 오름차순, 두 번째 원소: 내림차순, 세 번째 원소: 오름차순
    Arrays.sort(arr, (a, b) -> {
      if (a.x != b.x)
        return Integer.compare(a.x, b.x);
      else if (a.y != b.y)
        return Integer.compare(b.y, a.y);
      else
        return Integer.compare(a.z, b.z);
    });

    /*
     * 1 10 5
     * 1 10 5
     * 1 2 4
     * 2 5 7
     * 3 6 1
     * 3 6 2
     * 3 6 3
     * 3 5 7
     * 4 10 5
     * 4 3 1
     */

    // List 계열 Collection 생성
    List<Tuple> list = new ArrayList<>();

    list.add(new Tuple(1, 2, 3));
    list.add(new Tuple(2, 1, 6));
    list.add(new Tuple(4, 1, 5));
    list.add(new Tuple(3, 2, 2));
    list.add(new Tuple(5, 2, 4));
    list.add(new Tuple(1, 4, 3));
    list.add(new Tuple(3, 2, 6));
    list.add(new Tuple(5, 2, 7));
    list.add(new Tuple(1, 4, 6));
    list.add(new Tuple(1, 4, 5));

    // 1) List 인터페이스 메서드 사용
    // 첫 번째 원소(우선 정렬): 오름차순, 두 번째 원소: 내림차순, 세 번째 원소: 내림차순
    list.sort((a, b) -> {
      if (a.x != b.x)
        return Integer.compare(a.x, b.x);
      else if (a.y != b.y)
        return Integer.compare(b.y, a.y);
      else
        return Integer.compare(b.z, a.z);
    });

    /*
     * 1 4 6
     * 1 4 5
     * 1 4 3
     * 1 2 3
     * 2 1 6
     * 3 2 6
     * 3 2 2
     * 4 1 5
     * 5 2 7
     * 5 2 4
     */

    // 2) Collections 유틸리티 클래스 메서드 사용
    // 두 번째 원소(우선 정렬): 오름차순, 세 번째 원소: 내림차순, 첫 번째 원소: 오름차순
    Collections.sort(list, (a, b) -> {
      if (a.y != b.y)
        return Integer.compare(a.y, b.y);
      else if (a.z != b.z)
        return Integer.compare(b.z, a.z);
      else
        return Integer.compare(a.x, b.x);
    });

    /*
     * 2 1 6
     * 4 1 5
     * 5 2 7
     * 3 2 6
     * 5 2 4
     * 1 2 3
     * 3 2 2
     * 1 4 6
     * 1 4 5
     * 1 4 3
     */
  }
}
```
