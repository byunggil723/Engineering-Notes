# Math

### 1. 소수 n자리까지 내림

```java
double x = 3.14159;
double y = Math.floor(x * 100) / 100.0;

System.out.println(y); // 3.14
```

### 2. 소수 n자리까지 올림

```java
double x = 3.14159;
double y = Math.ceil(x * 100) / 100.0;

System.out.println(y); // 3.15
```

### 3. 소수 n자리까지 반올림

```java
double x = 3.14159;
double y = Math.round(x * 100) / 100.0;

System.out.println(y); // 3.14
```
