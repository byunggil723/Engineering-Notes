# Heap

### Min Heap (오름차순)

```Java
import java.util.*;

PriorityQueue<Integer> pq = new PriorityQueue<>();
// 최소 힙 (기본)
// 내부적으로 힙(완전 이진 트리) 구조를 사용하며, O(log N) 시간복잡도를 가진다.

// 삽입
pq.offer(3);
pq.offer(1);
pq.offer(2);

pq.peek();        // 조회 (최소값 반환, 제거 X, 비어있으면 null 반환) → 1

pq.poll();        // 삭제 (최소값 제거 후 반환, 비어있으면 null 반환) → 1

pq.size();
pq.isEmpty();

// 전체 순회 (순서 보장 X)
for (int x : pq) {
  System.out.println(x);
}
```

import java.util.*;

PriorityQueue<Integer> pq = new PriorityQueue<>(Collections.reverseOrder());
// 최대 힙 (Max Heap)
