# Map

### Map(HashMap)

```Java 
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Map<String, Integer> m = new HashMap<>();
    // key는 중복 불가, value는 중복 가능
    // HashMap은 가장 기본적인 구현체로, 해시 기반 O(1) 평균 시간복잡도를 가진다.

    m.put("A", 1); // 삽입 (같은 key로 put 하면 기존 value를 덮어 씀)
    m.put("B", 2);
    m.put("C", 3);

    m.get("A"); // 조회 → 1 (없으면 null 반환)
    m.containsKey("A"); // key 존재 여부 확인 → true

    m.put("C", 10); // 수정 (key가 "C"인 원소의 값을 10으로 덮어쓰기)
    m.replace("C", 12); // 수정 (key가 "C"인 원소의 값을 12로 변경)
    m.replace("D", 100); // key가 "D"인 원소가 없으므로 아무런 일도 일어나지 않음

    m.remove("A"); // 삭제
    m.size(); // 크기
    m.isEmpty(); // 비어있는지 확인

    // key 기반 순회 (순서 보장 X)
    // 삽입 순서를 유지하고 싶은 경우, Map<String, Integer> map = new LinkedHashMap<>(); 로 선언
    for (String key : m.keySet()) {
      System.out.println(key + " " + m.get(key));
    }

    // entry 기반 순회 (권장)
    for (Map.Entry<String, Integer> e : m.entrySet()) {
      System.out.println(e.getKey() + " " + e.getValue());
      // getKey(), getValue()는 Map.Entry 인터페이스 메서드
    }
  }
}
```

### TreeMap(정렬 Map)

#### 1\) 오름차순(기본)

```Java
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Map<Integer, String> m = new TreeMap<>();
    // key는 중복 불가, value는 중복 가능
    // TreeMap은 내부적으로 Red-Black Tree를 사용하여 key 기준으로 자동 정렬되는 Map 구현체로,
    // O(log N) 시간복잡도를 가진다.

    m.put(3, "C");
    m.put(1, "A");
    m.put(2, "B");

    m.get(1); // 조회 → "A" (없으면 null 반환)
    m.containsKey(2); // key 존재 여부 확인 → true

    m.remove(3); // 삭제
    m.size(); // 크기
    m.isEmpty(); // 비어있는지 확인

    // 순회
    for (int key : m.keySet()) {
      System.out.println(key + " " + m.get(key));
    }

    // entry 기반 순회 (권장)
    for (Map.Entry<Integer, String> e : m.entrySet()) {
      System.out.println(e.getKey() + " " + e.getValue());
    }

    /*
     * 1 A
     * 2 B
     */
  }
}
```

#### 2\) 내림차순

```java
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Map<Integer, String> m = new TreeMap<>((a, b) -> {
      return Integer.compare(b, a);
    });

    m.put(3, "C");
    m.put(1, "A");
    m.put(2, "B");

    m.get(1); // 조회 → "A" (없으면 null 반환)
    m.containsKey(2); // key 존재 여부 확인 → true

    m.remove(3); // 삭제
    m.size(); // 크기
    m.isEmpty(); // 비어있는지 확인

    // 순회
    for (int key : m.keySet()) {
      System.out.println(key + " " + m.get(key));
    }

    // entry 기반 순회 (권장)
    for (Map.Entry<Integer, String> e : m.entrySet()) {
      System.out.println(e.getKey() + " " + e.getValue());
    }

    /*
     * 2 B
     * 1 A
     */
  }
}
```

#### 3\) 복합 정렬

```java
import java.util.*;

class Pair {
  int x, y;

  Pair(int x, int y) {
    this.x = x;
    this.y = y;
  }
}

public class Main {
  public static void main(String[] args) throws Exception {
    Map<Pair, Integer> m = new TreeMap<>((a, b) -> {
      if (a.x != b.x)
        return Integer.compare(b.x, a.x);
      else
        return Integer.compare(a.y, b.y);
    }); // 사용자 정의 클래스 Pair는 내부적으로 비교 기준이 없으므로, TreeMap 선언 시 Comparator를 제공해야 한다.

    m.put(new Pair(1, 2), 1);
    m.put(new Pair(2, 1), 2);
    m.put(new Pair(2, 2), 3);
    m.put(new Pair(2, 3), 4);
    m.put(new Pair(4, 2), 5);

    // 조회
    System.out.println(m.get(new Pair(1, 2))); // 1
    System.out.println(m.get(new Pair(2, 2))); // 3

    // 존재 확인
    System.out.println(m.containsKey(new Pair(1, 2))); // true
    System.out.println(m.containsKey(new Pair(1, 3))); // false

    // 삭제
    System.out.println(m.remove(new Pair(2, 1))); // 2

    // 정렬된 상태로 반환
    for (Map.Entry<Pair, Integer> e : m.entrySet()) {
      System.out.println(e.getKey().x + " " + e.getKey().y + " " + e.getValue());
    }

    /*
     * 4 2 5
     * 2 2 3
     * 2 3 4
     * 1 2 1
     */
  }
}
```
