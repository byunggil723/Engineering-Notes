# Map

### Map(HashMap)

```Java 
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Map<String, Integer> map = new HashMap<>();
    // key는 중복 불가, value는 중복 가능
    // HashMap은 가장 기본적인 구현체로, 해시 기반 O(1) 평균 시간복잡도를 가진다.

    map.put("A", 1); // 삽입 (같은 key로 put 하면 기존 value를 덮어 씀)
    map.put("B", 2);

    map.get("A"); // 조회 → 1 (없으면 null 반환)
    map.containsKey("A"); // key 존재 여부 확인 → true

    map.remove("A"); // 삭제
    map.size(); // 크기
    map.isEmpty(); // 비어있는지 확인

    // key 기반 순회 (순서 보장 X)
    // 삽입 순서를 유지하고 싶은 경우, Map<String, Integer> map = new LinkedHashMap<>(); 로 선언
    for (String key : map.keySet()) {
      System.out.println(key + " " + map.get(key));
    }

    // entry 기반 순회 (권장)
    for (Map.Entry<String, Integer> entry : map.entrySet()) {
      System.out.println(entry.getKey() + " " + entry.getValue());
      // getKey(), getValue()는 Map.Entry 인터페이스 메서드
    }
  }
}
```

### TreeMap(정렬 Map)

#### 1\) 오름차순(기본)

```Java
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    Map<Integer, String> map = new TreeMap<>();
    // key는 중복 불가, value는 중복 가능
    // TreeMap은 내부적으로 Red-Black Tree를 사용하여 key 기준으로 자동 정렬되는 Map 구현체로,
    // O(log N) 시간복잡도를 가진다.

    map.put(3, "C"); // 삽입 (중복 삽입 → 무시됨)
    map.put(1, "A");
    map.put(2, "B");

    map.get(1); // 조회 → "A" (없으면 null 반환)
    map.containsKey(2); // key 존재 여부 확인 → true

    map.remove(3); // 삭제
    map.size(); // 크기
    map.isEmpty(); // 비어있는지 확인

    // 순회
    for (int key : map.keySet()) {
      System.out.println(key + " " + map.get(key));
    }

    // entry 기반 순회 (권장)
    for (Map.Entry<Integer, String> entry : map.entrySet()) {
      System.out.println(entry.getKey() + " " + entry.getValue());
    }

    /*
     * 1 A
     * 2 B
     */
  }
}
```

#### 3\) 복합 정렬

```java
import java.util.*;

class Pair {
  int x, y;

  Pair(int x, int y) {
    this.x = x;
    this.y = y;
  }
}

public class Main {
  public static void main(String[] args) throws Exception {
    Map<Pair, Integer> m = new TreeMap<>((a, b) -> {
      if (a.x != b.x)
        return Integer.compare(b.x, a.x);
      else
        return Integer.compare(a.y, b.y);
    });

    m.put(new Pair(1, 2), 1);
    m.put(new Pair(2, 1), 2);
    m.put(new Pair(2, 2), 3);
    m.put(new Pair(2, 3), 4);
    m.put(new Pair(4, 2), 5);

    // 조회
    System.out.println(m.get(new Pair(1, 2))); // 1
    System.out.println(m.get(new Pair(2, 2))); // 3

    // 존재 확인
    System.out.println(m.containsKey(new Pair(1, 2))); // true
    System.out.println(m.containsKey(new Pair(1, 3))); // false

    // 삭제
    System.out.println(m.remove(new Pair(2, 1))); // 2

    // 정렬된 상태로 반환
    for (Map.Entry<Pair, Integer> e : m.entrySet()) {
      System.out.println(e.getKey().x + " " + e.getKey().y + " " + e.getValue());
    }

    /*
     * 4 2 5
     * 2 2 3
     * 2 3 4
     * 1 2 1
     */
  }
}
```
