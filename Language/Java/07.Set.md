# Set

### Set(HashSet)

```Java
import java.util.*;

Set<Integer> set = new HashSet<>();
// 중복 허용 X, 순서 보장 X
// HashSet은 내부적으로 해시 기반으로 동작하며, 평균 O(1) 시간복잡도를 가진다.

set.add(1);          // 삽입
set.add(2);
set.add(2);          // 중복 삽입 → 무시됨

set.contains(1);     // 값 존재 여부 확인 → true

set.remove(1);       // 삭제
set.size();          // 크기
set.isEmpty();       // 비어있는지 확인

// 순회 (순서 보장 X)
// 삽입 순서를 유지하고 싶은 경우, Set<Integer> set = new LinkedHashSet<>(); 로 선언
for (int value : set) {
  System.out.println(value);
}
```

### TreeSet (정렬 Set)

```Java
import java.util.*;

Set<Integer> set = new TreeSet<>();
// 중복 허용 X, 자동 정렬 (오름차순)
// TreeSet은 내부적으로 Red-Black Tree를 사용하여 자동 정렬되는 Map 구현체로, 
// O(log N) 시간복잡도를 가진다.

set.add(3);
set.add(1);
set.add(2);

set.contains(2);     // true

set.remove(3);       // 삭제
set.size();          
set.isEmpty();       

// 정렬된 상태로 반환
for (int value : set) {
  System.out.println(value); // 1 2 3
}
```
