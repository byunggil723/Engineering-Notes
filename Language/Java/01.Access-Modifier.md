# 접근 제어자 (Access Modifier)

| 제어자       | 같은 클래스 | 같은 패키지 | 상속(다른 패키지) | 전체 |
| --------- | ------ | ------ | ---------- | -- |
| public    | O      | O      | O          | O  |
| protected | O      | O      | O          | X  |
| default   | O      | O      | X          | X  |
| private   | O      | X      | X          | X  |

#### * static

`static`은 멤버를 객체가 아니라 클래스 자체에 귀속시키는 일반 제어자이다.<br>
즉, static으로 선언된 멤버는 객체마다 생성되는 것이 아니라<br>
클래스 단위로 하나만 존재하여 프로그램 실행 동안 공유된다.<br>

따라서, 다음과 같은 프로그래머스의 실행 환경에서는 이전 테스트의 값이 누적될 수 있으니 주의해서 사용해야 한다.

```Java
new Solution().solution(test1);
new Solution().solution(test2);
new Solution().solution(test3);
```
