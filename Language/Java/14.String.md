# 문자열 메서드

## 1. String

`String`은 `immutable`한 객체이므로, 모든 메서드는 원본을 직접 수정하지 않고 새로운 문자열을 반환한다.

```java
int length()
// 길이 반환


char charAt(int i)
// index i의 문자 반환


String substring(int beginIndex) || String substring(int beginIndex, int endIndex)
// beginIndex부터 endIndex - 1까지의 부분 문자열을 반환


boolean equals(String str)
// 두 문자열이 같은지 반환, 반드시 String 타입의 인자를 대입


boolean contains(String str)
// 특정 문자열이 포함되어 있는지 반환, 반드시 String 타입의 인자를 대입


int indexOf(char c || String str)
// 특정 문자 혹은 문자열이 처음 등장하는 index를 반환, 없으면 -1 반환


int indexOf(char c || String str, int fromIndex)
// fromIndex부터 탐색하여 특정 문자 혹은 문자열이 처음 등장하는 index를 반환, 없으면 -1 반환


String replace(String target, String replacement)
// 특정 문자열을 모두 치환한 새로운 문자열을 반환, 반드시 String 타입의 인자를 대입


String[] split(String regex)
// 구분자(regex)를 기준으로 문자열을 나누어 배열로 반환, 반드시 String 타입의 인자를 대입
// 이때 split의 인자 regex는 일반 문자열이 아니라 정규표현식
"a.b.c".split(".")   // 의도대로 안 됨
"a.b.c".split("\\.") // 올바름

String s = "[a, b, c, d]";
String substr = s.substring(1, s.length() - 1); // a, b, c, d
String[] result = substr.split(", ");
for (int i = 0; i < result.length; i++) {
  System.out.print(result[i] + " ");
} // a b c d 
System.out.println(Arrays.toString(result)); // [a, b, c, d]


String[] split(String regex, int limit)
// 구분자(regex)를 기준으로 문자열을 나누되 개수를 제한하여 배열로 반환

String s = "[a, b, c, d]";
String substr = s.substring(1, s.length() - 1); // a, b, c, d
String[] result = substr.split(", ", 2);
for (int i = 0; i < result.length; i++) {
  System.out.println(result[i] + " ");
}
/*
 * a
 * b, c, d
 */
System.out.println(Arrays.toString(result)); // [a, b, c, d] (원소 2개: "a", "b, c, d")
```

## 2. StringBuilder

`StringBuilder`는 `mutable`한 객체이므로, 문자열을 조작하는 메서드는 새로운 객체를 만들지 않고 원본 자체를 수정한다.

```java
int length()
// 길이 반환


char charAt(int i)
// index i의 문자 반환


String substring(int beginIndex) || String substring(int beginIndex, int endIndex)
// beginIndex부터 endIndex - 1까지의 부분 문자열을 반환, 원본 수정이 아님에 주의!


StringBuilder sb1 = new StringBuilder("abc");
StringBuilder sb2 = new StringBuilder("abc");
System.out.println(sb1.toString().equals(sb2.toString())); // true
// StringBuilder의 내용 비교 방식


int indexOf(String str)
// 문자열이 처음 등장하는 index를 반환, 없으면 -1 반환, 반드시 String 타입의 인자를 대입


int indexOf(String str, int fromIndex)
// fromIndex부터 탐색하여 특정 문자열이 처음 등장하는 index를 반환, 없으면 -1 반환, 반드시 String 타입의 인자를 대입


StringBuilder replace(int start, int end, String str)
// start부터 end - 1까지를 str로 교체

StringBuilder sb = new StringBuilder("abcde");
sb.replace(1, 4, "ZZ");
System.out.println(sb); // aZZe


StringBuilder sb = new StringBuilder("a, b, c");
String[] arr = sb.toString().split(", ");
// StringBuilder의 split 방식


StringBuilder append(...)
// 문자열 맨 뒤에 값을 이어 붙임

sb.append("This is ");
System.out.println(sb); // This is 
sb.append(100);
sb.append("% ");
System.out.println(sb); // This is 100%
sb.append(true);
System.out.println(sb); // This is 100% true
sb.append('.');
System.out.println(sb); // This is 100% true.


void setCharAt(int i, char c)
// index i의 문자를 c로 변경, 반드시 char 타입의 인자를 대입


StringBuilder insert(int idx, ...)
// index idx 위치에 값을 삽입

sb.insert(0, "This is ");
System.out.println(sb); // This is
sb.insert(sb.length(), true);
System.out.println(sb); // This is true
sb.insert(8, 100);
System.out.println(sb); // This is 100true
sb.insert(11, "% ");
System.out.println(sb); // This is 100% true
sb.insert(sb.length(), '.');
System.out.println(sb); // This is 100% true.


StringBuilder delete(int start, int end)
// index start부터 end - 1까지의 문자를 삭제


StringBuilder deleteCharAt(int i)
// index i의 문자 하나를 삭제


StringBuilder reverse()
// 문자열을 뒤집음


String toString()
// StringBuilder 내용을 String으로 변환하여 반환
```
