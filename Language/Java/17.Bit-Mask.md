# Bit Mask

정수의 각 비트에 특정 의미를 부여하여, 여러 개의 선택 여부나 상태를 하나의 값으로 표현한 것

## 기본 비트 연산

### 1. OR (`|`)

하나라도 1인 경우, 1

```java
public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1001;
    int B = 0b0101;

    /*
     *   1001
     * | 0101
     * =======
     *   1101
     */

    String bitA = Integer.toBinaryString(A);
    String bitB = Integer.toBinaryString(B);

    System.out.println("Decimal_A : " + A); // 9
    System.out.println("Binary_A : " + bitA); // 1001

    System.out.println("-----------------------------");

    System.out.println("Decimal_B : " + B); // 5
    System.out.println("Binary_B : " + bitB); // 0101

    System.out.println("-----------------------------");

    int result = A | B;

    String bitResult = Integer.toBinaryString(result);

    System.out.println("Decimal_result : " + result); // 13
    System.out.println("Binary_result : " + bitResult); // 1101
  }
}
```

### 4. NOT (`~`)

```java
public class Main {

    System.out.println("Decimal_A : " + A); // 9

  }
}
```

### 5. Shift (`<<`, `>>`)

`<<`  : 비트를 왼쪽으로 이동시키고, 오른쪽 빈칸은 `0`으로 채움<br>
`>>`  : 비트를 오른쪽으로 이동시키고, 왼쪽 빈칸은 **부호 비트**(양수면 0, 음수면 1)로 채움<br>
`>>>` : 비트를 오른쪽으로 이동시키고, 왼쪽 빈칸은 `0`으로 채움

```java
public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1001;

    String bitA = Integer.toBinaryString(A);

    System.out.println("Decimal_A : " + A); // 9
    System.out.println("Binary_A : " + bitA); // 1001

    System.out.println("-----------------------------");

    int resultLeftMove = A << 2;

    String bitResultLeftMove = Integer.toBinaryString(resultLeftMove);

    System.out.println("Decimal_result : " + resultLeftMove); // 36
    System.out.println("Binary_result : " + bitResultLeftMove); // 100100

    System.out.println("-----------------------------");

    int resultRightMove = A >> 2;

    String bitResultRightMove = Integer.toBinaryString(resultRightMove);

    System.out.println("Decimal_result : " + resultRightMove); // 2
    System.out.println("Binary_result : " + bitResultRightMove); // 0010
  }
}
```

## 비트 마스킹

### 1. 공집합, 접체 집합

```java
public class Main {
  public static void main(String[] args) throws Exception {
    int A, B;

    A = 0; // 공집합

    B = (1 << 7) - 1; // 원소의 개수가 최대 7개인 집합 표현이 가능 (1이 총 7개)

    String bitA = Integer.toBinaryString(A);
    String bitB = Integer.toBinaryString(B);

    System.out.println("Decimal_A : " + A); //0
    System.out.println("Binary_A : " + bitA); // 0

    System.out.println("-----------------------------");

    System.out.println("Decimal_B : " + B); // 127
    System.out.println("Binary_B : " + bitB); // 1111111
  }
}
```

### 2. 원소 추가

`A = A | (1 << k)`:  k + 1번째 비트를 1로 설정, 이때 가장 오른쪽 비트의 인덱스는 `0`

```java
public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1001;

    String bitA = Integer.toBinaryString(A);

    System.out.println("Decimal_A : " + A); // 9
    System.out.println("Binary_A : " + bitA); // 1001

    System.out.println("-----------------------------");

    A = A | (1 << 2);

    /*
     * index       3210
     * bit A       1001
     * (1 << 2)  | 0100
     * =================
     * result      1101
     */

    bitA = Integer.toBinaryString(A);

    System.out.println("Decimal_A : " + A); // 13
    System.out.println("Binary_A : " + bitA); // 1101
  }
}
```

### 3. 원소 삭제

`A = A & ~(1 << k)`:  k + 1번째 비트를 0으로 설정, 이때 가장 오른쪽 비트의 인덱스는 `0`

```java
public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1001;

    String bitA = Integer.toBinaryString(A);

    System.out.println("Decimal_A : " + A); // 9
    System.out.println("Binary_A : " + bitA); // 1001

    System.out.println("-----------------------------");

    A = A & ~(1 << 3);

    /*
     * index       3210
     * bit A       1001
     * ~(1 << 3) & 0111
     * =================
     * result      0001
     */

    bitA = Integer.toBinaryString(A);

    System.out.println("Decimal_A : " + A); // 1
    System.out.println("Binary_A : " + bitA); // 0001
  }
}
```

### 4. 특정 원소 포함 여부 확인

`if ((A & (1 << 2)) == 0)`: 자바에서는 if 조건식에 int 값을 직접 사용할 수 없으므로, 비교 연산을 통해 boolean 값으로 만들어야 한다.

```java
public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1001;

    String bitA = Integer.toBinaryString(A);

    System.out.println("Decimal_A : " + A); // 9
    System.out.println("Binary_A : " + bitA); // 1001

    System.out.println("-----------------------------");

    if ((A & (1 << 2)) == 0) {
      System.out.println("The 2nd element is not in the set.");
    } else {
      System.out.println("The 2nd element is in the set.");
    }

    if ((A & (1 << 3)) == 0) {
      System.out.println("The 3rd element is not in the set.");
    } else {
      System.out.println("The 3rd element is in the set.");
    }

    // The 2nd element is not in the set.
    // The 3rd element is in the set.
  }
}
```

### 5. 특정 비트 상태 반전

`A = A ^ (1 << k)`: k + 1번째 비트를 반전한다.

```java
import java.io.*;
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1001;

    String bitA = Integer.toBinaryString(A);

    System.out.println("Decimal_A : " + A); // 9
    System.out.println("Binary_A : " + bitA); // 1001

    System.out.println("-----------------------------");

    A = A ^ (1 << 3);

    /*
     * index       3210
     * bit A       1001
     * (1 << 3)  ^ 1000
     * =================
     * result      0001
     */

    if ((A & (1 << 3)) == 0) {
      System.out.println("The 3rd element is not in the set.");
    } else {
      System.out.println("The 3rd element is in the set.");
    }

    A = A ^ (1 << 3);

    /*
     * index       3210
     * bit A       0001
     * (1 << 3)  ^ 1000
     * =================
     * result      1001
     */

    if ((A & (1 << 3)) == 0) {
      System.out.println("The 3rd element is not in the set.");
    } else {
      System.out.println("The 3rd element is in the set.");
    }

    // The 3rd element is not in the set.
    // The 3rd element is in the set.
  }
}
```

### 6. 집합의 연산

#### 3\) 차집합

`A & ~B`: A - B<br>
`B & ~A`: B - A<br>

```java
import java.io.*;
import java.util.*;

public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1001;
    int B = 0b0101;

    String bitA = Integer.toBinaryString(A);
    String bitB = Integer.toBinaryString(B);

    System.out.println("Decimal_A : " + A); // 9
    System.out.println("Binary_A : " + bitA); // 1001

    System.out.println("-----------------------------");

    System.out.println("Decimal_B : " + B); // 5
    System.out.println("Binary_B : " + bitB); // 0101

    System.out.println("-----------------------------");

    int result1 = A & ~B; // A - B

    /*
     * result 1
     *   1001 (A)
     * & 1010 (~B, B = 0101)
     * ==========
     *   1000 (A - B)
     */

    int result2 = B & ~A; // B - A

    /*
     * result 2
     *   0101 (B)
     * & 0110 (~A, A = 1001)
     * ==========
     *   0100 (B - A)
     */

    String bitResult1 = Integer.toBinaryString(result1);

    System.out.println("Decimal_result1 : " + result1); // 8
    System.out.println("Binary_result1 : " + bitResult1); // 1000

    System.out.println("-----------------------------");

    String bitResult2 = Integer.toBinaryString(result2);

    System.out.println("Decimal_result2 : " + result2); // 4
    System.out.println("Binary_result2 : " + bitResult2); // 0100
  }
}
```

### 4\) 대칭차

`A ^ B`: A와 B 중 정확히 하나에만 포함된 원소들의 집합 (= A △ B)

```java
public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1001;
    int B = 0b0101;

    String bitA = Integer.toBinaryString(A);
    String bitB = Integer.toBinaryString(B);

    System.out.println("Decimal_A : " + A); // 9
    System.out.println("Binary_A : " + bitA); // 1001

    System.out.println("-----------------------------");

    System.out.println("Decimal_B : " + B); // 5
    System.out.println("Binary_B : " + bitB); // 0101

    System.out.println("-----------------------------");

    int result = A ^ B; // A △ B

    /*
     * result
     *   1001 (A)
     * ^ 0101 (B)
     * ==========
     *   1100 (A △ B)
     */

    String bitResult = Integer.toBinaryString(result);

    System.out.println("Decimal_result : " + result); // 12
    System.out.println("Binary_result : " + bitResult); // 1100
  }
}
```

### 10. 모든 부분 집합 순회

공집합을 제외한 모든 부분 집합 순회

```java
public class Main {
  public static void main(String[] args) throws Exception {
    int A = 0b1010;

    String bitA = Integer.toBinaryString(A);

    System.out.println("Decimal_A : " + A); // 10
    System.out.println("Binary_A : " + bitA); // 1010

    System.out.println("-----------------------------");

    for (int subset = A; subset != 0; subset = A & (subset - 1)) {
      System.out.println(Integer.toBinaryString(subset));
    }

    // 1010
    // 1000
    // 0010
  }
}
```
