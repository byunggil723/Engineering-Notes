# Binary Search

### 1. binary search

이분 탐색 (탐색 대상이 되는 배열은 반드시 오름차순 정렬이 되어 있어야 한다.)

```java
static int binarySearch(int[] arr, int target) {
  int left = 0;
  int right = arr.length - 1;
  while (left <= right) {
    int mid = (left + right) / 2;
    if (arr[mid] == target) {
      return mid; // 찾은 위치 반환
    } else if (arr[mid] < target) {
      left = mid + 1; // 오른쪽 탐색
    } else {
      right = mid - 1; // 왼쪽 탐색
    }
  }
  return -1; // 없음
}
```

### 2. lower bound

처음으로 `target` 이상이 나오는 위치를 반환

```java
static int lowerBound(int[] arr, int target) {
  int left = 0;
  int right = arr.length; // [left, right)
  while (left < right) {
    int mid = (left + right) / 2;
    if (arr[mid] >= target) {
      right = mid;
    } else {
      left = mid + 1;
    }
  }
  return left;
}
```

### 3. upper bound

처음으로 `target` 초과가 나오는 위치 반환

```java
static int upperBound(int[] arr, int target) {
  int left = 0;
  int right = arr.length; // [left, right)
  while (left < right) {
    int mid = (left + right) / 2;
    if (arr[mid] > target) {
      right = mid;
    } else {
      left = mid + 1;
    }
  }
  return left;
}
```
