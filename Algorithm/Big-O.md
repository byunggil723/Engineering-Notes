# 빅오 표기법

빅오 표기의 정의는 다음과 같다.

어떤 알고리즘의 실행 시간 함수가 $T(N)$일 때, 충분히 큰 $N$에 대해<br>
$T(N) ≤ C⋅f(N)$<br>
을 만족하는 상수 $C, N_{0}$ 가 존재하면,<br>
$T(N)=O(f(N))$<br>
라고 한다.

여기서 중요한 점은

1. Big-O는 정확한 실행 시간을 말하기 위함이 아니다.
2. Big-O는 입력이 무한히 커질 때의 성장률만을 다룬다.

예를 들어:<br>
$T_{1}(N) = 10N_{2}$<br>
$T_{2}(N) = 0.001N_{2}$

이 둘은 실제 실행 시간은 1만 배 차이지만,<br>
$T_{1}(N), T_{2}(N) ∈ O(N^2)$<br>
로 같은 클래스로 분류한다.

즉, 빅오 표기법은 어떤 알고리즘의 정확한 실행 시간을 계산하기 위한 도구가 아니라,<br>
현재 알고리즘이 어떤 시간 복잡도 계층으로 분류되는지를 대표적으로 나타내기 위한 표기법이다.
