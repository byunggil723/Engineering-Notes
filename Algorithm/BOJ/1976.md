# 여행 가자

> 문제

동혁이는 친구들과 함께 여행을 가려고 한다. 한국에는 도시가 N개 있고 임의의 두 도시 사이에 길이 있을 수도, 없을 수도 있다.<br>
동혁이의 여행 일정이 주어졌을 때, 이 여행 경로가 가능한 것인지 알아보자. 물론 중간에 다른 도시를 경유해서 여행을 할 수도 있다.<br>
예를 들어 도시가 5개 있고, A-B, B-C, A-D, B-D, E-A의 길이 있고, 동혁이의 여행 계획이 E C B C D 라면 E-A-B-C-B-C-B-D라는 여행경로를 통해 목적을 달성할 수 있다.

도시들의 개수와 도시들 간의 연결 여부가 주어져 있고, 동혁이의 여행 계획에 속한 도시들이 순서대로 주어졌을 때 가능한지 여부를 판별하는 프로그램을 작성하시오.<br>
같은 도시를 여러 번 방문하는 것도 가능하다.

> 입력

첫 줄에 도시의 수 N이 주어진다. N은 200이하이다. 둘째 줄에 여행 계획에 속한 도시들의 수 M이 주어진다. M은 1000이하이다.<br>
다음 N개의 줄에는 N개의 정수가 주어진다. i번째 줄의 j번째 수는 i번 도시와 j번 도시의 연결 정보를 의미한다. 1이면 연결된 것이고 0이면 연결이 되지 않은 것이다. A와 B가 연결되었으면 B와 A도 연결되어 있다.<br>
마지막 줄에는 여행 계획이 주어진다. 도시의 번호는 1부터 N까지 차례대로 매겨져 있다.

> 출력

첫 줄에 가능하면 YES 불가능하면 NO를 출력한다.

> 풀이

주어진 경로들을 유니온-파인드 알고리즘을 통해 모두 엮었을 때, 각 노드들이 갈 수 있는 대표 종착점이 같다면 어떤 순서와 개수로 주어져도 반드시 해당 경로를 만들 수 있음을 활용하는 문제.

그 이유는 간단하다. 유니온-파인드로 같은 집합에 포함된 노드들은 모두 동일한 대표 노드(루트)에 연결되어 있으며, 각 노드는 대표 노드까지 도달 가능한 경로가 반드시 존재하므로,<br>
이 집합을 대표 노드를 중심으로 하나의 성형(Star Topology) 연결 요소로 보면, 어떤 순서로든 도시들을 이동하는 경로를 구성할 수 있기 때문이다.

### 정답 코드

```C++
#include <iostream>
#include <vector>

using namespace std;

int N = 0, M = 0;
vector<int> parent;

int find_set(int x)
{
  if (parent[x] == x)
    return x;

  return parent[x] = find_set(parent[x]);
}

void union_set(int a, int b)
{
  if (a == b)
    return;

  int representative_of_a = find_set(a);
  int representative_of_b = find_set(b);

  if (representative_of_a > representative_of_b)
    parent[representative_of_a] = representative_of_b;
  else
    parent[representative_of_b] = representative_of_a;
}

int main()
{
  cin.tie(NULL);
  ios::sync_with_stdio(false);

  cin >> N;

  cin >> M;

  for (int i = 0; i <= N; i++)
    parent.push_back(i);

  int x = 0;
  for (int i = 1; i <= N; i++)
  {
    for (int j = 1; j <= N; j++)
    {
      cin >> x;

      if (x == 1)
        union_set(i, j);
    }
  }

  cin >> x;
  int prev_value = find_set(x);
  bool possible = true;
  for (int i = 1; i < M; i++)
  {
    cin >> x;

    if (prev_value != find_set(x))
    {
      possible = false;
      break;
    }
  }

  if (possible)
    cout << "YES";
  else
    cout << "NO";
}
```
