# 다각형의 면적

> 문제

2차원 평면상에 N(3 ≤ N ≤ 10,000)개의 점으로 이루어진 다각형이 있다.<br>
이 다각형의 면적을 구하는 프로그램을 작성하시오.

> 입력

첫째 줄에 N이 주어진다.<br>
다음 N개의 줄에는 다각형을 이루는 순서대로 N개의 점의 x, y좌표가 주어진다.<br>
좌표값은 절댓값이 100,000을 넘지 않는 정수이다.

> 출력

첫째 줄에 면적을 출력한다.<br>
면적을 출력할 때에는 소수점 아래 둘째 자리에서 반올림하여 첫째 자리까지 출력한다.

> 풀이

가우스 면적 공식(신발끈 공식)을 활용한다.

### 정답 코드

```C++
#include <cmath>
#include <iomanip>
#include <iostream>
#include <vector>

using namespace std;

int N = 0;
vector<double> x;
vector<double> y;
double result = 0;

int main() {
  cin >> N;

  int a = 0, b = 0;
  for (int i = 0; i < N; i++) {
    cin >> a >> b;

    x.push_back(a);
    y.push_back(b);
  }
  x.push_back(x[0]);
  y.push_back(y[0]);

  double left_to_right = 0;
  double right_to_left = 0;
  for (int i = 0; i < N; i++) {
    left_to_right = left_to_right + (x[i] * y[i + 1]);
    right_to_left = right_to_left + (x[i + 1] * y[i]);
  }

  result = abs(left_to_right - right_to_left) / 2;

  cout << fixed << setprecision(1) << result; // 소수 둘째 자리에서 반올림
}
```
