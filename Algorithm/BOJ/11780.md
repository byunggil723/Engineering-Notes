# 플로이드 2

> 문제

n(1 ≤ n ≤ 100)개의 도시가 있다. 그리고 한 도시에서 출발하여 다른 도시에 도착하는 m(1 ≤ m ≤ 100,000)개의 버스가 있다.
각 버스는 한 번 사용할 때 필요한 비용이 있다.<br>
모든 도시의 쌍 (A, B)에 대해서 도시 A에서 B로 가는데 필요한 비용의 최솟값을 구하는 프로그램을 작성하시오.

> 입력

첫째 줄에 도시의 개수 n이 주어지고 둘째 줄에는 버스의 개수 m이 주어진다. 그리고 셋째 줄부터 m+2줄까지 다음과 같은 버스의 정보가 주어진다.<br>
먼저 처음에는 그 버스의 출발 도시의 번호가 주어진다. 버스의 정보는 버스의 시작 도시 a, 도착 도시 b, 한 번 타는데 필요한 비용 c로 이루어져 있다.<br>
시작 도시와 도착 도시가 같은 경우는 없다. 비용은 100,000보다 작거나 같은 자연수이다.

> 출력

먼저, n개의 줄을 출력해야 한다. i번째 줄에 출력하는 j번째 숫자는 도시 i에서 j로 가는데 필요한 최소 비용이다.<br>
만약, i에서 j로 갈 수 없는 경우에는 그 자리에 0을 출력한다.

그 다음에는 n×n개의 줄을 출력해야 한다. i×n+j번째 줄에는 도시 i에서 도시 j로 가는 최소 비용에 포함되어 있는 도시의 개수 k를 출력한다.<br>
그 다음, 도시 i에서 도시 j로 가는 경로를 공백으로 구분해 출력한다. 이때, 도시 i와 도시 j도 출력해야 한다. 만약, i에서 j로 갈 수 없는 경우에는 0을 출력한다.

> 풀이

플로이드-워셜 알고리즘의 모든 노드 쌍 간 최단거리뿐 아니라 경로까지 복원하는 문제다.<br>
경로 복원 과정은 다음과 같다.

1\) 경로 복원을 위해 history[i][j] 배열을 두고, 이를 i에서 j로 가기 위해 거쳐야 할 처음 노드로 정의한다.

2\) 초기 입력값에 대해 간선 i → j 가 존재하면 history[i][j] = j 로 초기화한다. (없으면 0 또는 -1)

3\) 플로이드-워셜 수행 중, 경유 노드 k를 추가로 허용했을 때 최단거리가 갱신되면<br>
$$dist[i][j] > dist[i][k] + dist[k][j]$$

일 때,

$$
dist[i][j] = dist[i][k] + dist[k][j]
$$

$$
history[i][j] = history[i][k]
$$

로 갱신한다.<br>
즉, i → j 최단 경로가 i → k → j 형태로 바뀌면, i에서 출발해서 처음으로 가야 할 노드는 우선적으로 i → k 경로의 첫 노드가 되므로 history[i][k]를 대입한다.

4\) 이 과정이 반복되어 결과적으로 history[i][j] 에는 i에서 j로 가기 위해 가장 먼저 거쳐야 하는 다음 노드가 저장된다.

5\) 따라서 경로를 복원할 때는 start_node = i에서 출발하여, start_node = history[start_node][j] 를 반복하며 방문 노드(start_node)를 차례대로 기록하면 i → j 경로를 얻을 수 있다.<br>
이때 history[i][j]가 0 또는 -1이면 경로가 존재하지 않으며, start_node == j가 될 때까지 진행하므로 마지막 j도 경로에 포함된다.

### 정답 코드

```C++
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int n = 0, m = 0;
long long INF = 1e12;
vector<vector<long long>> dist;
vector<vector<int>> history;

int main()
{
  cin.tie(NULL);
  ios::sync_with_stdio(false);

  cin >> n >> m;

  dist.assign(n + 1, vector<long long>(n + 1, INF));
  history.assign(n + 1, vector<int>(n + 1, 0));

  int a = 0, b = 0;
  long long c = 0;
  for (int i = 0; i < m; i++)
  {
    cin >> a >> b >> c;

    dist[a][b] = min(dist[a][b], c);
    history[a][b] = b;
  }

  for (int i = 1; i <= n; i++)
  {
    dist[i][i] = 0;
  }

  for (int k = 1; k <= n; k++)
  {
    for (int i = 1; i <= n; i++)
    {
      if (dist[i][k] == INF)
        continue;
      for (int j = 1; j <= n; j++)
      {
        if (dist[k][j] == INF)
          continue;

        if (dist[i][j] > dist[i][k] + dist[k][j])
        {
          dist[i][j] = dist[i][k] + dist[k][j];
          history[i][j] = history[i][k];
        }
      }
    }
  }

  for (int i = 1; i <= n; i++)
  {
    for (int j = 1; j <= n; j++) // i에서 j로 가는 경로가 존재하지 않는 경우
    {
      if (dist[i][j] == INF)
        cout << 0 << ' ';
      else
        cout << dist[i][j] << ' ';
    }
    cout << '\n';
  }

  vector<vector<long long>>().swap(dist);
  // clear()는 size만 0으로 만들고, capacity(할당된 메모리)는 남아 있을 수 있다.
  // 반면 빈 벡터와 swap하면 dist가 들고 있던 버퍼(원소 저장을 위한 힙 메모리)가 임시 객체로 넘어가고,
  // dist는 빈 벡터가 가지고 있던 버퍼를 받게 되어 size = 0, capacity = 0 이 된다.
  // 이후 임시 객체가 즉시 소멸하면서, 임시 객체가 들고 있던 원래 dist의 버퍼가 해제되므로
  // 결과적으로 dist의 capacity까지 포함해 메모리가 반납된다.

  for (int i = 1; i <= n; i++)
  {
    for (int j = 1; j <= n; j++)
    {
      if (history[i][j] == 0) // i에서 j로 가는 경로가 존재하지 않는 경우
      {
        cout << 0 << '\n';
        continue;
      }

      int start_node = i, end_node = j;
      vector<int> path(1, start_node);

      while (start_node != end_node)
      {
        path.push_back(history[start_node][end_node]);
        start_node = history[start_node][end_node];
      }

      cout << path.size() << ' ';
      for (int k = 0; k < path.size(); k++)
      {
        cout << path[k] << ' ';
      }
      cout << '\n';

      path.clear();
    }
  }
}
```
